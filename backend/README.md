# GEN-AI + FULL STACK WEBDEVELOPMENT 
## Project Guideline 
### For backend:
-	First of all, I have to create node.js application –  npm init -y 
-	Create server – npm install express mongoose dotenv 
-	For server start command: npx nodemon .\server.js
-	For hashing and token cookie parse : npm i bcryptjs jsonwebtoken cookie-parser
-	Important website we must follow:  search on google: jwtsecretes.com 

### For frontend:
-	For frontend: npm create vite@latest client --template react
-	For frontend dependencies: npm i axios react-router-dom
-	For frontend start command: npm run dev
-	For deployment: npm i -g vercel
-	For deployment: vercel login
-	For deployment: vercel --prod

### For database:
-	For database: mongodb atlas
-	For database connection: npm install mongoose
-   For database connection: create .env file and add the connection string in it.
-   For database connection: create a config folder and create db.js file in it and add the connection code in it.
-   For database connection: import the db.js file in server.js file and call the function to connect to the database.
